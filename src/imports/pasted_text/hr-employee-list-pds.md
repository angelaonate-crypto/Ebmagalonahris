Update the **HR Staff Employee List page** so that it matches the same **Employee Directory layout and Personal Data Sheet (PDS – CS Form 212) structure** used in the HR Officer and Department Head interfaces.

Do not change the HRIS design system such as colors, typography, spacing, sidebar layout, header style, card components, or icons.

The goal is to make the **Employee List consistent across all roles**.

---

EMPLOYEE DIRECTORY LAYOUT

Use the same **two-column Employee Directory layout** used in the HR Officer and Department Head views.

LEFT COLUMN

Employee List

Display a searchable list of employees including:

Employee Name
Employee ID
Position Title
Department
Employment Status

Include search and filter options.

Selecting an employee should update the profile panel on the right.

---

RIGHT COLUMN

Employee Profile

Display the employee record using the **Personal Data Sheet (PDS – CS Form 212) structure**.

Use the same section order used in other roles.
