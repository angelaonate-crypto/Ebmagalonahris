Update the **Department Head interface pages** so that they match the same UI design system used in the **HR Officer and Employee interfaces**.

Do not change the color theme, typography, spacing, or component styling. Maintain the same HRIS design system.

The goal is to make the Department Head interface visually consistent with the rest of the system.

---

GLOBAL LAYOUT

Apply the same layout structure used in the HR Officer and Employee dashboards.

Layout should include:

Left Sidebar Navigation
Top Header Bar
Main Content Area

Sidebar width, icon style, spacing, and card components should match the HR Officer interface.

Header layout should match the other roles.

Header elements:

Page title
Search bar
Notification bell
User profile avatar
Dropdown menu (Profile / Settings / Logout)

Notifications must appear only in the header.

---

Reports

Left Column

Employee list with:

Employee Name

Right Column

Employee Profile and History.
