Add an **Employee View Dashboard** to the HR Management System.

The system has two roles:
• HR Officer
• Employee

When an employee logs in successfully (via password or facial verification), they should be redirected to the **Employee Dashboard**.

---

EMPLOYEE DASHBOARD LAYOUT

Create a clean dashboard layout for employees.

Structure:
• Left sidebar navigation
• Top header bar
• Main content area

Top Header:
• Page title (Employee Dashboard)
• Notification icon
• Profile avatar
• Dropdown menu (Profile / Settings / Logout)

Sidebar Navigation:

Dashboard
My Attendance
Leave Requests
Payslips
Announcements
My Profile
Settings
Logout

---

EMPLOYEE DASHBOARD OVERVIEW

Create an overview page showing personal information and summaries.

Top Cards:
• Today's Attendance Status
• Total Leave Balance
• Pending Leave Requests
• Upcoming Holidays

Widgets:
• Recent announcements
• Attendance summary
• Upcoming events or holidays

---

MY ATTENDANCE PAGE

Create an attendance tracking page for employees.

Features:
• Calendar view of attendance
• Attendance status (Present / Late / Absent)
• Daily check-in and check-out times
• Total hours worked

Optional feature:
• Face verification check-in button

---

LEAVE REQUEST PAGE

Allow employees to request leave.

Leave Request Form Fields:
• Leave type
• Start date
• End date
• Reason
• Submit request button

Below the form display a **Leave History Table**.

Columns:
• Leave type
• Start date
• End date
• Status (Pending / Approved / Rejected)

---

PAYSLIPS PAGE

Create a payroll page where employees can access salary records.

Table columns:
• Month
• Salary amount
• Bonus
• Deductions
• Net salary
• Download payslip

---

ANNOUNCEMENTS PAGE

Display HR announcements for employees.

Each announcement card should include:
• Title
• Message
• Date
• Posted by HR

---

MY PROFILE PAGE

Employee profile management page.

Fields:
• Profile photo
• Full name
• Email
• Phone number
• Department
• Position
• Date joined

Include an **Edit Profile button**.

---

DESIGN STYLE

The Employee dashboard should match the overall HR system design:

• Clean SaaS dashboard style
• Rounded cards
• Soft shadows
• Consistent spacing
• Neutral background
• Blue or purple accent colors

Ensure the **Employee Dashboard layout matches the HR Dashboard design style but with simplified controls and limited permissions**.
